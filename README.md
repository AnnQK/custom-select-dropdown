## Custom select / dropdown menu

Native select value is updated and connected with custom component.

#### Built with:

- React
- Typescript
- Simple css

#### Run it locally:

```bash
npm install
```

```bash
npm run dev
```

##### You can hide native select with adding this to styles.css

<pre><code>.native-select{
    position: absolute;
    opacity: 0;
}</code></pre>
